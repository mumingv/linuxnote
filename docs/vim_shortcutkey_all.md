# vim快捷键汇总

###  

|快捷键     |作用                                       |
|-----------|-------------------------------------------|
|a          |在当前光标所在字符后进入Insert模式         |
|A          |在行尾进入Insert模式                       |
|b          |反向定位到单词头部                         |
|B          |反向定位到非空字符串头部                   |
